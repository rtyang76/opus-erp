# Opus-ERP 用户手册

> 本文档介绍 Opus-ERP 系统的使用方法。

---

## 1. 系统登录

### 1.1 登录页面

1. 打开浏览器，访问系统地址（如 http://localhost）
2. 输入用户名和密码
3. 点击"登录"按钮

### 1.2 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 系统管理员 |

**注意：首次登录后请立即修改密码！**

### 1.3 退出登录

1. 点击右上角用户头像
2. 选择"退出登录"
3. 确认退出

## 2. 系统管理

### 2.1 用户管理

**路径：** 系统管理 → 用户管理

**功能：**
- 查看用户列表
- 新增用户
- 编辑用户信息
- 启用/禁用用户
- 重置密码
- 删除用户

**操作步骤：**

1. **新增用户**
   - 点击"新增用户"按钮
   - 填写用户信息（用户名、密码、昵称等）
   - 选择角色
   - 点击"确定"保存

2. **编辑用户**
   - 在用户列表中点击"编辑"按钮
   - 修改用户信息
   - 点击"确定"保存

3. **重置密码**
   - 在用户列表中点击"重置密码"按钮
   - 确认重置
   - 新密码为：123456

### 2.2 角色管理

- 删除角色

### 2.3 菜单管理

## 4. 采购管理

### 4.1 采购订单

**路径：** 采购管理 → 采购订单

**功能：**
- 查看采购订单列表
- 新增采购订单
- 审核采购订单
- 取消采购订单

**操作步骤：**

1. **新增采购订单**
   - 点击"新增采购订单"按钮
   - 选择供应商
   - 添加订单明细（物料、数量、单价）

2. **审核采购订单**

## 9. 常见问题

### 9.1 登录问题

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码。

**Q: 账号被禁用怎么办？**
A: 联系系统管理员启用账号。

### 9.2 操作问题

**Q: 无法删除数据？**
A: 已审核或有关联数据的记录无法删除。

**Q: 库存不足无法出库？**
A: 请检查库存数量，或联系仓库管理员。

### 9.3 系统问题

**Q: 页面加载缓慢？**
A: 检查网络连接，或联系技术支持。

**Q: 数据显示异常？**
A: 尝试刷新页面，或联系技术支持。
